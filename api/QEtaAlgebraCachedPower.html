The domain QEtaAlgebraCachedPower(C, A) behaves like the given QEtaAlgebra A, but to each of its elements x a power x^n is never computed twice, but rather stored and then accessed from the cache. Although the implementation of formulated with weaker conditions on the input parameter, we assume that A forms a C algebra.
