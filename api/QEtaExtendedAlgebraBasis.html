QEtaExtendedAlgebraBasis(C, L, F, AB) is a domain holding the special structure of an algebra basis with elements that can be used during a reduction. This structure is useful to access basis elements as if they were from the domain L. The structure is just another view on an element of QEtaAlgebraBasis(C, F) such that it looks like QEtaAlgebraBasis(C, L) by always taking first(x) of any element of F.
