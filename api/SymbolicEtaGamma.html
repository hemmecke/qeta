SymbolicEtaGamma collects data for the expansion of eta_delta(gamma tau). See eqref{eq:eta_delta-m-lambda(gamma*tau)}.

eta: (PositiveInteger, PositiveInteger, Matrix Integer) -> %

eta(mm, delta, gamma) represents the expansion of eta_delta(gamma tau) in terms of q = exp(2 pi i tau). It is the same as eta(mm, delta, 1, 0, gamma). See eqref{eq:eta_delta-m-lambda-gamma-tau} for m=1, lambda=0.

eta: (PositiveInteger, PositiveInteger, NonNegativeInteger, Matrix Integer) -> %

eta(delta, m, lambda, gamma) returns eta(delta, delta, m, lambda, gamma)

eta: (PositiveInteger, PositiveInteger, PositiveInteger, NonNegativeInteger, Matrix Integer) -> %

eta(mm, delta, m, lambda, gamma) represents the expansion of $eta_{delta,m,lambda}(gamma tau)$ in terms of $q = exp(2 pi i tau)$. See eqref{eq:eta_delta-m-lambda-gamma-tau}. We require that c>0 in gamma=matrix[[a,b],[c,d]].
