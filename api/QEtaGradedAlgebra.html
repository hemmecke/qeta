QEtaGradedAlgebra(C) lists the minimal signatures for running the functions in the QEtaPackage. Mathematically seen is it a C-algebra that is Z-graded. Simplest example is a univariate polynomial ring over C, but since it is expected that respective domains will be created on the fly, we list here only the functions that are actually used in QEtaSambaPackage.
