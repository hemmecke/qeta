algclos.spad line 83 [edit on github]

Implement an ACF according to cite{Steel:AlgebraicallyClosedFields:2010}, i.e. the algebraic closure of rational numbers. In contrast to the domain AlgebraicNunber in FriCAS, this domain certifies that that zero tests are corrct. Let tst(a, b)==(x:=sqrt(a*b);y:=sqrt(a)*sqrt(b);zero?(x-y) or zero?(x+y)), then tst(2::AN,3::AN) returns false whereas tst(2::DAN,3::DAN) gives true.
