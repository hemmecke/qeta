identity(sspec, rspec, m, orb, rspecss, rel, xab) returns a structure that represents all information about the q-expansion of $F_{s,r,m,t}$ at infinity, see Definition of $F_{s,r,m,t}$ in qeta.tex, Equation~eqref{eq:F_s-r-m-t(tau)}. The relations is then c*F=sum_{i}(rel(i)*etaQuotient(rspecs(i))) where F corresponds to the modular function $F_{s,r,m,t}$ in question. t is an element of orb. At the same time this data structure serves as a container for a generalized eta-quotient identity for $F_{bar{s},bar{r},m,t}(tau)$.
