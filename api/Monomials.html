  • dim: NonNegativeInteger
  • D: DirectProductCategory (dim, NonNegativeInteger )
  • vl: List Symbol

Monomials(dim, D, vl) behaves exactly like D, i.e. is an AbelianMonoid, but prints its elements in a multiplicative form. For example, if dim = 2, vl = [A,B] and x::Vector(NNI) = [2,3], then the element x (coerced to OutputForm) looks like A^2*B^3.
