QEtaQuotientSpecificationss4ti2 computes a monoid basis of (specifications for) eta-quotients.

etaQuotientMonoidSpecifications: (PositiveInteger, List Integer) -> List QEtaSpecification

etaQuotientMonoidSpecifications(m, idivs) computes the matrixLigozatModularGamma0(m, idivs)$QEtaAuxiliaryPackage and finds a basis of the solution space. It computes the same as etaQuotientSpecifications(m, idivs, 0).

etaQuotientMonoidSpecifications: PositiveInteger -> List QEtaSpecification

etaQuotientMonoidSpecification(m) returns specifications of eta-quotients that correspond to the formula (16) from cite{Radu:RamanujanKolberg:2015}, i.e. (together with 1) they describe the monoid E^infty(m). Same as etaQuotientsMonoidSpecifications(m, [1..n-1]) where n is the number of divisors of n.

etaQuotientSpecifications: (PositiveInteger, List Integer, Integer) -> List QEtaSpecification

etaQuotientSpecifications(m, idivs, weight) computes etaQuotientSpecifications(m, idivs, rhs, weight) for rhs := [0 for i in idivs].

etaQuotientSpecifications: (PositiveInteger, List Integer, List Integer, Integer) -> List QEtaSpecification

etaQuotientSpecifications(m, idivs, rhs, weight) computes zsolve4ti2(m, idivs, concat(rhs, [weight]) and chooses one particular solution and adds it to all homogeneous and free solution vectors. The opposite vectors of the free solutions are also taken into account.

generalizedEtaQuotientMonoidSpecifications: (PositiveInteger, List List Integer) -> List QEtaSpecification

generalizedEtaQuotientMonoidSpecifications(nn, idxs) returns specifications of generalized eta-quotients that (together with 1) generate the (multiplicative) monoid of all generalized eta-quotients (corresponding to the indices given by idxs) that are modular functions for $Gamma_1(nn)$ and have at most a pole at infinity. If idxs=[], then it is equivalent to the indices generalizedEtaFunctionIndices(nn).

generalizedEtaQuotientMonoidSpecifications: PositiveInteger -> List QEtaSpecification

generalizedEtaQuotientMonoidSpecifications(nn) returns generalizedEtaQuotientMonoidSpecifications(nn, []).

generalizedEtaQuotientSpecifications: (PositiveInteger, List List Integer, Integer) -> List QEtaSpecification

generalizedEtaQuotientSpecifications(nn, idxs, grd) returns specifications for generatilzed eta-quotients with a pole of pole-order grd>0 at infinity and no pole at any other cusp.

zsolve4ti2: (PositiveInteger, List Integer, List Integer) -> Record(zinhom: List Vector Integer, zhom: List Vector Integer, zfree: List Vector Integer)

zsolve4ti2(m, idivs, rhs) computes n:=#idivs, mat:=matrixLigozatModularGamma0(m,idivs)$QEtaAuxiliaryPackage and the column vector b whose first entries are given by rhs. Note that the (n+1)-th entry corresponds to the weight. See in the specification of matrixLigozatModularGamma0 about the meaning of the other rows of the matrix.
