SymbolicGeneralizedEtaQuotientLambdaGamma¶

qetaquotsymb.spad line 1659 [edit on github]

SymbolicGeneralizedEtaQuotientLambdaGamma holds data to compute a generalized eta-quotient expansion of $g_{bar{r},m,lambda}(gammatau)$. See eqref{eq:g_rtilde-m-lambda(gamma*tau)}. eqref{eq:g_rbar(gamma*tau)}.

generalizedEtaQuotient: (PositiveInteger, List List Integer, PositiveInteger, NonNegativeInteger, Matrix Integer) -> %

generalizedEtaQuotient(mm, rbar, m, lambda, gamma) represents the expansion of $g_{bar{r},m,lambda}(gammatau)$. see eqref{eq:g_rbar-m-lambda(gamma*tau)}.

generalizedEtaQuotient(rspec, m, lambda, gamma) represents the expansion of $g_{bar{r},m,lambda}(gammatau)$. see eqref{eq:g_rbar-m-lambda(gamma*tau)} where rbar is given by rspec.
