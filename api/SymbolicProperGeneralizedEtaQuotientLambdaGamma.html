properGeneralizedEtaQuotient: (PositiveInteger, List List Integer, PositiveInteger, NonNegativeInteger, Matrix Integer) -> %

properGeneralizedEtaQuotient(nn,rtilde,m,lambda,gamma) represents the expansion of $g_{tilde{r},m,lambda}(gammatau)$. see eqref{eq:g_rtilde-m-lambda(gamma*tau)}.

qExponent: % -> Fraction Integer

If x=properGeneralizedEtaQuotient(nn,rtilde,m,lambda,gamma), then qExponent(x) returns the argument of the third product of eqref{eq:g_rtilde(gamma*tau)}, i.e. $gensum frac{r_{delta,g} u_delta}{2} P_2left(frac{a_delta g}{delta}right)$.

unityExponent: % -> Fraction Integer

If x=properGeneralizedEtaQuotient(nn,rtilde,m,lambda,gamma), then unityExponent(x) returns the argument of the first product of eqref{eq:g_rtilde(gamma*tau)}.
