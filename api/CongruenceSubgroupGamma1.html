CongruenceSubgroupGamma1 implements functions connected to the congruence subgroup $Gamma_1(N)$. https://www.wstein.org/books/modform/modform/dimension_formulas.html

cusps: PositiveInteger -> List Cusp

cusps(m) returns representative for all the (inequivalent) cusps for Gamma_1(m) sorted ascendingly. cusp(0,1) will be the first and cusp(1,0)=infinity() will be the last.

cuspToMatrix: (PositiveInteger, Cusp) -> Matrix Integer

cuspToMatrix(m, cusp) returns a matrix gamma=[[a,b],[c,d]] corresponding to the cusp of Gamma1(m).

cuspWidth: (PositiveInteger, Integer, Integer) -> Integer

cuspWidth(m, a, c) returns the width of the cusp a/c of Gamma_1(m)

dimensionOfCuspForms: (PositiveInteger, Integer) -> NonNegativeInteger

dimensionOfCuspForms(m, w) computes the dimension of S_w(Gamma_1(m)). See https://www.wstein.org/books/modform/modform/dimension_formulas.html

dimensionOfEisensteinSubspace: (PositiveInteger, Integer) -> NonNegativeInteger

dimensionOfEisensteinSubspace(m, w) computes the dimension of E_w(Gamma_1(m)), the Eisenstein subspace of modular forms See https://www.wstein.org/books/modform/modform/dimension_formulas.html

dimensionOfModularForms: (PositiveInteger, Integer) -> NonNegativeInteger

dimensionOfModularForms(m, w) computes the dimension of M_w(Gamma_1(m)). See https://www.wstein.org/books/modform/modform/dimension_formulas.html

doubleCosetRepresentatives: PositiveInteger -> List Matrix Integer

doubleCosetRepresentatives(nn) returns a list of double coset representatives of $Gamma_1(nn) backslash Gamma_1(1) / SL_2(setZ)_infty$ where $SL_2(setZ)_infty$ are matrices of the form [[1,h],[0,1]] with h being an integer.

equivalentCusps?: (PositiveInteger, Cusp, Cusp) -> Boolean

equivalentCusps?(nn, a/c, u/w) returns true iff the cusp a/c is equivalent to u/w modulo the action of Gamma1(nn). Two cusps a/c and u/w are equivalent modulo the action of Gamma1(N) if either (1) c = w (mod N) and a = u (mod gcd(c, N) or (2) c = -w (mod N) and a = -u (mod gcd(c, N). For c=0, we interpret a/c as the cusp infinity. See Proposition 2.27 in the PhD thesis of William Stein: https://www.wstein.org/thesis/.

genus: PositiveInteger -> NonNegativeInteger

genus(level) returns the genus of the congruence subgroup Gamma_1(level). We use formulae given at https://www.wstein.org/books/modform/modform/dimension_formulas.html See Section âModular Formas for $Gamma_1(N)$â in cite{Stein:ModularForms:2007}.

index: PositiveInteger -> PositiveInteger

index(n) computes the index of Gamma_1(n) in SL2Z.

member?: (Matrix Integer, PositiveInteger) -> Boolean

member?(mat, n) returns true if mat is an element of $Gamma_1(n)$, i.e. if entry (2,1) is a multiple of n and entries (1,1) and (2,2) are congruent 1 (mod n). Here it is assumed that determinant(mat)=1.

normalizeCusp: (PositiveInteger, Cusp) -> Cusp

normalizeCusp(nn, cusp) normalizes the cusp to another (equivalent modulo Gamma1(nn)) cusp (u:w) such that numerator and denominator are positive and small. Exceptions are 0=(0:1) and infinity()=(1:0).

nu2: PositiveInteger -> Integer

nu2(n) returns the number of $Gamma_1(n)$ inequivalent elliptic points of order 2. This is mu_{1,2}(n) of https://www.wstein.org/books/modform/modform/dimension_formulas.html and known to be zero if n > 2.

nu3: PositiveInteger -> Integer

nu3(n) returns the number of $Gamma_1(n)$ inequivalent elliptic points of order 3. This is mu_{1,3}(n) of https://www.wstein.org/books/modform/modform/dimension_formulas.html and known to be zero if n > 3.

numberOfCusps: PositiveInteger -> PositiveInteger

numberOfCusps(m) returns the number of cusps of $Gamma_1(m)$. See https://www.wstein.org/books/modform/modform/dimension_formulas.html

projectiveIndex: PositiveInteger -> PositiveInteger

projectivIndex(n) computes the index of the image of Gamma_1(n) in PSL2Z. It holds: if odd?(n), then projectiveIndex(n)=index(n), otherwise projectiveIndex(n)=index(n)/2.

rightCosetRepresentatives: (PositiveInteger, PositiveInteger) -> List Matrix Integer

rightCosetRepresentatives(nn, mm) returns a list of right coset representatives of $Gamma_1(mm*nn) backslash Gamma_1(mm)$.

rightCosetRepresentatives: PositiveInteger -> List Matrix Integer

rightCosetRepresentatives(nn) returns rightCosetRepresentatives(nn, 1). It is a list of right coset representatives of $Gamma_1(nn) backslash Gamma_1(1)$.

width: (PositiveInteger, Cusp) -> Integer

width(m, cusp) returns the width of the cusp for Gamma_1(m). If cusp=(a:c), the width is independent of a and given by m/gcd(c,m) except when m=4 and gcd(c,m)=2 where the width is 1. See Equation (2.49) in cite{Chen+Du+Zhao:FindingModularFunctionsRamanujan:2019} which is in Corollary 4 (2) of cite{Cho+Koo+Park:ArithmeticRamanujanGoellnitzGordon:2009}.
