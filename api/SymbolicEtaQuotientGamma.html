SymbolicEtaQuotientGamma¶

qetaquotsymb.spad line 488 [edit on github]

SymbolicEtaQuotientGamma holds data to compute an eta quotient expansion of $p_{r,m,t}(gamma tau)$. See eqref{eq:p_r-m-t(gamma*tau)}.

divisors: % -> List PositiveInteger

divisors(x) returns the divisors of level(x) that were given at creation time of x.

elt: (%, NonNegativeInteger) -> SymbolicEtaQuotientLambdaGamma

x.lambda returns the data corresponding to the respective lambda.

etaQuotient: (PositiveInteger, List Integer, Matrix Integer) -> %

etaQuotient(mm, r, gamma) returns etaQuotient(mm,r,1,0,gamma) and represents the expansion of $p_r(gamma tau)$ in terms of x = exp(2 pi i tau/w) where w=width(m, c) and gamma=matrix[[a,b],[c,d]] represents the cusp a/c. $r$ is indexed by the divisors of $mm$.

etaQuotient: (PositiveInteger, List Integer, PositiveInteger, Integer, Matrix Integer) -> %

etaQuotient(mm, r, m, t, gamma) represents the expansion of $p_{r,m,t}(gamma tau)$ where $r$ is indexed by the divisors of $mm$.

etaQuotient: (QEtaSpecification, Matrix Integer) -> %

etaQuotient(rspec, gamma) returns etaQuotient(rspec,1,0,gamma) and represents the expansion of $p_r(gamma tau)$ in terms of x = exp(2 pi i tau/w) where w=width(m, c) and gamma=matrix[[a,b],[c,d]] represents the cusp a/c. $r$ is given by rspec.

etaQuotient: (QEtaSpecification, PositiveInteger, Integer, Matrix Integer) -> %

etaQuotient(rspec, m, t, gamma) represents the expansion of $p_{r,m,t}(gamma tau)$ where $r$ is given by rspec.

minimalRootOfUnity: % -> PositiveInteger

minimalRootOfUnity(x) returns the smallest positive integer n such that the expansion of the function $p_{r,m,t}(gammatau)$ corresponding to x=etaQuotient(mm,divs,r,m,t,gamma) (neglecting the (c*tau+d)^* factor) lives in Q[w][[z]] where w is a n-th root of unity and z a fractional q power.

one?: % -> Boolean

one?(x) returns true if the eta-quotient corresponding to x represents 1. This is the case if one?(specification(x)), i.e. if the representation is trivial.

qExponent: % -> Fraction Integer

qExponent(x) returns the order of the q-expansion in terms of the original q. Note that this exponent is only a lower bound for the q-expansion. The coefficient corresponding to this q-power may be zero.

specification: % -> QEtaSpecification

If x=etaQuotient(spec,…), then specification(x) returns spec.

unityExponent: % -> Fraction Integer

unityExponent(x) returns -(24*t + hat{r})/(24*m) if x = etaQuotient(mm,r,m,t,gamma).
