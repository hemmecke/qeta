SymbolicEtaQuotientGamma holds data to compute an eta quotient expansion of g_r(gamma tau). See eqref{eq:g_r(gamma*tau)}.

etaQuotient: (PositiveInteger , List PositiveInteger , List Integer , Matrix Integer ) -> %
etaQuotient(m, divs, r, gamma) represents the expansion of $g_r(gamma tau) = prod_{delta in divisors(s)} eta(delta gamma tau)^{r_{delta}}$ in terms of x = exp(2 pi i tau/w) where w=width(m, c) and gamma=cuspToMatrix(m, a/c) for a cusp a/c.

minimalRootOfUnity: % -> PositiveInteger
minimalRootOfUnity(e) returns the smallest positive integer n such that the expansion of the function g_r(gamma tau) corresponding to e=etaQuotient(m, divs, r, gamma) (neglecting the (ctau+d)^* factor) lives in Q[w][[x]] where w is a n-th root of unity.
modular?: % -> Boolean
modular?() returns true if the eta quotient is a modular function for Gamma0(level x), i.e, it returns modularConditions?(level x, exponents x)$QEtaKolberg.
qExponent: % -> Fraction Integer
qExponent(e) returns 24 times the order of the expansion of e in q = exp(2 pi i tau) while neglecting the (c tau+d) factor. It corresponds to 24 times the exponent of the fourth product of eqref{eq:g_r(gamma*tau)}.
rationalPrefactor: % -> Fraction Integer
rationalPrefactor(x) returns the square of the second product in eqref{eq:g_r(gamma*tau)}.
unityPower: % -> Fraction Integer
unityPower(e) returns sum_delta rdelta*(vdelta+kappa_delta)/24. It corresponds to the third product of eq:g_r(gamma*tau).
xExponent: % -> Integer
xExponent(e) returns 24 times the order of the expansion of e in x = exp(2 pi i tau/w) where w is the width of the cusp while neglecting the (c tau+d) factor. It corresponds to 24 times the exponent of the fourth product of eqref{eq:g_r(gamma*tau)}.
