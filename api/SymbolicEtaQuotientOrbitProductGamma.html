SymbolicEtaQuotientOrbitProductGamma¶

qetaquotsymb.spad line 682 [edit on github]

SymbolicEtaQuotientOrbitProductGamma holds data to compute an eta-quotient expansions of $P_{r, m, t}(gamma tau)$. See eqref{eq:P_r-m-t(gamma*tau)}.}

elt: (%, NonNegativeInteger) -> SymbolicEtaQuotientGamma

x.u returns the data corresponding to the respective orbit element u.

etaQuotient: (PositiveInteger, List Integer, PositiveInteger, NonNegativeInteger, Matrix Integer) -> %

etaQuotient(mm, r, m, t, gamma) represents the expansion of $P_{r, m, t}(gamma tau)$. $r$ is indexed by the divisors of $mm$.

etaQuotient: (QEtaSpecification, PositiveInteger, NonNegativeInteger, Matrix Integer) -> %

etaQuotient(rspec, m, t, gamma) represents the expansion of $P_{r, m, t}(gamma tau)$. $r$ is given by rspec.

exponents: % -> List Integer

exponents(x) returns the list of exponents corresponding to all divisors. Zeros at the end of the list are removed.

gamma: % -> Matrix Integer

gamma(x) returns the transformation corresponding to x.

from SetCategory

latex: % -> String

level: % -> PositiveInteger

level(x) returns the level of the eta-quotient.

minimalRootOfUnity: % -> PositiveInteger

minimalRootOfUnity(x) returns lcm [minimalRootOfUnity(x.u) for u in orbit(x)] where $rhat = sum_delta delta r_delta$

multiplier: % -> PositiveInteger

multiplier(x) returns the subsequence multiplier. Returns m.

offset: % -> NonNegativeInteger

offset(x) returns the subsequence offset. Returns t.

one?: % -> Boolean

one?(x) returns true if the eta-quotient orbit product corresponding to x represents 1. This is true if one?(specification(x)), i.e. the representation is trivial.

orbit: % -> List NonNegativeInteger

orbit(x) returns $modularOrbit{r, m, t}$ where r=exponents(x), m=multiplier(x), t=offset(x). See definition of orbit in qetacofactorspace.spad

qExponent: % -> Fraction Integer

qExponent(x) returns reduce(_+, [qExponent(x.k) for k in orbit(x)]) Note that this exponent is only a lower bound for the q-expansion. The coefficient corresponding to this q-power may be zero.

specification: % -> QEtaSpecification

If x=etaQuotient(spec,…), then specification(x) returns spec.
